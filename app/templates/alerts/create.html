💡 How Job Alerts Work

We'll search for new jobs matching your criteria and send you email notifications. You can create multiple alerts for different job types!

📋 Example Alert

Name: "Remote Software Engineer Jobs"
Keywords: "software engineer, developer, python"
Location: "Remote"
Frequency: Daily digest at 9 AM

✓ Give your alert a memorable name so you can identify it later

✓ Separate multiple keywords with commas
✓ Be specific to get better matches (e.g., "senior python developer" vs just "developer")

✓ Leave blank for all locations
✓ Use "Remote" for remote-only jobs

✓ Select which platforms to search (at least one required)

💡 Recommended: Daily for active job searching, Weekly for passive searching

💡 Pro Tips

  • Create multiple alerts for different job types or locations
  • Use specific keywords to reduce irrelevant matches
  • Start with daily frequency and adjust based on volume
  • You can edit or pause alerts anytime from the alerts dashboard
